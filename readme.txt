
GINV_tester: основной файл для тестирования работы, обрабатывает файлы JSON, вычисляет базис и сохраняет полученные решения в массив и таблицы .CSV, .XLSX.

json.zip - архив с файлами JSON, содержащими тестовые данные для решения.

tests.xlsx и tests.csv - результаты тестирования GINV в табличном виде.

ginv_tests.json - результаты тестирования в массиве JSON.
